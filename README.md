# ElectViz: Election Data Visualization for Media

## 🗳️ Project Overview

This project was completed as part of our internship with **Infosys Springboard Virtual Internship 6.0**.
**ElectionViz** is an interactive data visualization project developed using **Power BI** to analyze and present insights from the **India General Election 2024 dataset**.

The project transforms raw election data into **interactive dashboards** that help users understand **state-wise trends, constituency-level competition, and party performance**. It demonstrates how data visualization can simplify complex election data into **clear and meaningful insights**.

## 🚀 Key Features

* 📊 **Interactive Dashboards:**
  Dynamic visuals showing seat distribution, vote share, and party performance.

* 🌍 **State-Level Analysis:**
  Analyze election results across different states and identify dominant parties.

* 🏛️ **Constituency-Level Insights:**
  Detailed view of candidate performance, vote share, and competition.

* 🧠 **Party Performance Analysis:**
  Comparison of parties based on seats won and vote share.

* ⚙️ **KPI Metrics:**
  Key indicators like total seats, alliance-wise seats, and vote distribution.

* 📈 **Alliance Analysis:**
  Comparison between **NDA, INDIA, and Other parties**.

## 🛠️ Tools & Technologies

* **Power BI:** Dashboard creation and visualization
* **Power Query Editor:** Data cleaning and transformation
* **DAX (Data Analysis Expressions):** Custom measures and calculations
* **Excel:** Dataset preparation

## 📊 Dashboard Modules

### 1️⃣ State Analysis

* State-wise vote share distribution
* Party comparison within states
* Top constituencies by votes

### 2️⃣ Constituency Analysis

* Top candidates by votes
* Party-wise vote share
* Winner vs runner-up comparison

### 3️⃣ Party Performance Analysis

* Seats won by each party
* Contribution to alliances
* Comparison of major parties

### 4️⃣ Overview Analysis

* KPI Cards (NDA, INDIA, Others)
* Donut Chart (Alliance Distribution)
* Gauge Chart (Total Seats – 543)
* Bar Chart (Seats by Party)
